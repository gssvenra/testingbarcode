How To
Purpose of the App
To demonstrate how to scan a Barcode using a device with a camera.
Prerequisites
•
SmartUX Studio installed and setup.
•
Physical device with a camera.
Steps to run the Test app
1.
Install on a physical device
2.
Open the App
3.
Click the Barcode button, it will launch the camera
4.
Once the scan is completed, the text will display in the Text Input boxes on the screen.
5.
Tap the Clear Barcode button to clear and start again.
